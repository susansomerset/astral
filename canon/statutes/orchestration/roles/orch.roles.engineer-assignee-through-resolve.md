# Statement

The implementing engineer remains Linear assignee through resolve and child User Testing (prep-uat assigns Susan on the parent only).

## Rationale

Rollcall and parent sub-issue lists show who built each child.

## Examples

### Conforming

- After Review Posted, Ada stays assignee while moving to User Testing.

### Violating

- Child is reassigned to Susan at Review Posted before UAT rollup.
